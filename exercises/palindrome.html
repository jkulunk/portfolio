Palindrome - A JavaScript Exercise

Problem: Check if the given text is a palindrome. A palindrome is a word or phrase that reads the same backward as forward. Examples: Racecar, madam, nurses run.

Solution: Split the text into an array removing any spaces. Then loop over the array in reverse order and copy each character into a new array. Finally, loop over both arrays comparing the characters in each. If the characters do not match, exit the loop and output the result (return false). If the loop reaches the end without any mismatch then the text is a palindrome (return true).
